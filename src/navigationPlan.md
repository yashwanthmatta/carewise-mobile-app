# Mobile Navigation Plan

1. Welcome and consent
2. Login / create account
3. Patient profile
4. Dashboard
5. Upload report
6. Report analysis result
7. Diet and habit recommendations
8. Doctor search
9. Insurance matching
10. Subscription checkout
11. Medication organizer
12. Weekly check-in
13. Notification settings
14. Clinician review status
15. Privacy, export, and delete settings

Mobile must keep emergency safety available even if AI generation is unavailable.

Each screen needs loading, error, empty, offline, and slow-network states before app-store submission.
