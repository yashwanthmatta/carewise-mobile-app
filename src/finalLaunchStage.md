# CareWise AI Final Launch Stage

This is the final-stage path from working MVP to real iOS and Android launch.

## Complete In Code

- Mobile signup, login, refresh, secure saved session, and logout.
- Mobile consent recording and patient profile sync.
- Mobile PDF/image/text report picker.
- Mobile multipart report upload to the production backend.
- Backend report storage, analysis, lab trends, privacy export, and deletion foundation.
- Web MVP for investor demos and user testing.

## Must Finish Before Public Store Release

- Run iOS TestFlight and Android internal testing with real devices.
- Verify Cloudflare R2 or S3 production storage secrets in Render.
- Add production OpenAI key only in backend environment variables.
- Add human-reviewed Privacy Policy, Terms, Medical Disclaimer, and Data Deletion process.
- Complete Apple App Privacy and Google Play Data Safety forms from the real data flows.
- Collect feedback from at least one licensed clinician before using real patient decisions.

## Safe Product Positioning

CareWise AI helps users understand uploaded health reports in simple language and prepare questions for licensed professionals. It must not claim to diagnose, cure, prescribe, or replace professional care.
